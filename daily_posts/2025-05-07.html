Starting to write about the Heisenberg algebra/group

I’m trying to understand this from Kirrilov’s book (here \(\mathfrak{h}\) design the Heisenberg Lie algebra and \(H\) is the associated Lie group):

Exercise 2. Describe all closed connected subgroups in \(H\) and determine which of them are normal.

Hint. All such subgroups have the form \(A = \exp(\mathfrak{a})\) where \(\mathfrak{a} \subset \mathfrak{h}\) is a Lie subalgebra. Normal subgroups in \(H\) correspond to ideals in \(\mathfrak{h}\).

Here I’ll start writing about everything I know about the Heisenberg group/algebra and Lie groups/algebras in general, starting really basic (I’ll definitely not write everything today but that’s not the goal: I’m back to writing once a day no matter what!)

Let’s write

The Heisenberg algebra is the Lie algebra generated by three elements \(X, Y, Z\) and described by these relations: \[[X,Y] = Z \text{ and } [\,\cdot\,, Z] = 0\] (the other relations are implicit given the anti-symmetry)

We have a \(3 \times 3\) matrix realisation of \(\mathfrak{h}\) : \[X = \begin{pmatrix} 0 & 1 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix} \hspace{5pt} Y = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & 1 \\ 0 & 0 & 0 \end{pmatrix} \hspace{5pt} Z = \begin{pmatrix} 0 & 0 & 1 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}\] (the Lie bracket being effectively given by the commutator \([A,B] = AB - BA\) for \(A,B \in H\))

We can define the exponential as usual on \(\h \subset \Mat_n(\R)\) as a Taylor series: \[\exp(A) = \sum_{k \in \N} \frac{1}{k!}A^k\]

We easily see that the series converges for any \(A=xX+yY+zZ \in \h\) and we can even compute it since every triple product vanishes. Let’s do it.

The exponential is defined as \[\exp(A) = I_3 + (xX + yY + zZ) + \frac{1}{2}(xX + yY + zZ)^2 + \frac{1}{6}(xX + yY + zZ)^3 + \ldots\] but we have \[\begin{align*} (xX + yY + zZ)^2 &= x^2X^2 + xyXY + xzXZ + yxYX + y^2Y^2 + yzYZ + zxZX + zyZY + z^2Z^2 \\ &= 0 + xyZ + 0 + 0 + 0 + 0 + 0 + 0 + 0 \\ &= Z \end{align*}\] so in the end we get \[\begin{align*} \exp(A) &= I_3 + (xX + yY + zZ) + \frac{1}{2}xyZ + \underbrace{\frac{1}{6}(xX + yY + zZ)Z}_{=0} + \underbrace{\ldots}_{=0} \\ &= \begin{pmatrix} 1 & x & z+\frac{xy}{2} \\ 0 & 1 & y \\ 0 & 0 & 1 \end{pmatrix} \end{align*}\]

Let’s call it a day. It will start getting more interesting tomorrow.
